About Me

Hi, I'm Shukri, a front-end developer with a passion for clean, accessible, and responsive design. I specialize in HTML and CSS, building websites that not only look good but work well on all devices. I'm constantly improving my skills in web standards, layout techniques like Flexbox and Grid, and responsive design principles. I enjoy transforming ideas into browser-ready websites and collaborating with others to bring digital experiences to life. Currently, I'm expanding my knowledge into JavaScript and modern frameworks to take my front-end skills to the next level. When I'm not coding, you’ll find me exploring new design trends, contributing to small web projects, or experimenting with CSS animations.

Services

HTML

I write clean, semantic HTML5 code with a focus on accessibility and SEO best practices. I'm experienced in building structured layouts, creating forms, and embedding multimedia content. My approach ensures that websites are well-organized and easy to maintain.

CSS

I use modern CSS to craft responsive, visually engaging websites. I'm skilled in Flexbox and CSS Grid for layout, and I enjoy adding smooth transitions and animations to enhance user experience. I'm also familiar with CSS variables, the BEM naming convention, and working from design tools like Figma.

JavaScript

I use JavaScript to bring interactivity and functionality to web pages. I’m confident with DOM manipulation, event handling, and working with APIs using fetch and JSON. I follow modern ES6+ syntax and structure my code in a clean, modular way. I’m also exploring frameworks like React to deepen my front-end development skills.
